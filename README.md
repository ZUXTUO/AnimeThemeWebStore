# AnimeThemeWebStore

## 中文

### 项目特点
- **简易部署**：旨在让用户快速上手，主要还是在教育方面使用。
- **推荐环境**：  
  - **Web 服务器**：`Apache 2.4.39`
  - **PHP 版本**：`7.3.4nts`
  - **数据库**：`MySQL 5.7.26`

### 使用说明
#### Windows 用户
如果觉得手动配置麻烦，可以使用小皮面板来快速部署。

#### Linux 用户
请根据自己的需求自行设置服务器环境。

#### 数据库导入
1. 在 MySQL 中创建一个名为 `shop` 的数据库。
2. 将 `./database/shop.sql` 文件导入该数据库。
3. 修改数据库账号和密码，更新 `./php/db.php` 文件中的相关配置。

### 注意事项
此项目仅为教学用途，可能不适合直接用于商业运营。请谨慎使用！
